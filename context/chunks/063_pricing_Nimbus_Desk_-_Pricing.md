# Nimbus Desk — Pricing
Source: pricing

Nimbus Desk Pricing Tiers:
- **Free**: Free
  - Email ticketing
  - Basic shared inbox
  - Standard web widget
- **Starter**: $20/user/mo ($16/user/mo billed annually)
  - Omnichannel inbox
  - Basic automations
  - Standard SLA policies
  - Custom ticket views
- **Professional**: $50/user/mo ($40/user/mo billed annually)
  - Advanced routing
  - Custom dashboards
  - Multiple SLA policies
  - CSAT surveys
- **Enterprise**: Custom / Contact sales
  - Nimbus SSO integration
  - Advanced data security
  - Dedicated success manager
  - Sandbox environment
Add-ons:
- Additional Storage: $10 / 100GB / month — Expand your file attachment storage capacity.
