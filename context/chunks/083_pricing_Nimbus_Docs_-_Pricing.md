# Nimbus Docs — Pricing
Source: pricing

Nimbus Docs Pricing Tiers:
- **Free**: Free
  - Real-time editing
  - Basic templates
  - Up to 50 documents
- **Starter**: $8/user/mo ($6/user/mo billed annually)
  - Unlimited documents
  - Advanced search
  - Guest sharing
- **Professional**: $15/user/mo ($12/user/mo billed annually)
  - Unlimited version history
  - Custom templates
  - Page analytics
  - Team workspaces
- **Enterprise**: Custom / Contact sales
  - SSO integration
  - Advanced security
  - Dedicated support
  - Data loss prevention
Add-ons:
- Nimbus Docs AI: $9 / user / month — Generative AI writing assistant to summarize notes and draft content.
