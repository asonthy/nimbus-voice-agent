# Nimbus Endpoint — Features
Source: products

Nimbus Endpoint Key Features:
- Cross-platform device management (Windows, Mac, iOS, Android)
- Remote lock and wipe capabilities
- Automated patch management and OS updates
- Next-generation antivirus (NGAV)
- Bring Your Own Device (BYOD) separation
- Application deployment and blacklisting
- USB device control and encryption enforcement
- Real-time device health monitoring
- Geolocation tracking for lost devices
- Automated compliance reporting
