# Nimbus Docs — FAQs
Source: faqs

Nimbus Docs FAQs:
Q: What does Nimbus Docs cost?
A: Nimbus Docs is free for basic use, $8 per user per month for Starter, $15 per user per month for Professional, and custom for Enterprise.
Q: Is there a free trial?
A: Yes, you can try the Professional plan free for 14 days.
Q: What is the refund policy for Nimbus Docs?
A: We provide a 30-day money-back guarantee on all paid subscriptions if it does not meet your expectations.
Q: Can I import my existing documents from other platforms?
A: Yes, you can import standard formats like Word, Markdown, and text files directly into your workspaces.
