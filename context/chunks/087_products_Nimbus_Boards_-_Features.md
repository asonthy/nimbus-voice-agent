# Nimbus Boards — Features
Source: products

Nimbus Boards Key Features:
- Infinite canvas digital whiteboards
- Customizable kanban board layouts
- Sticky notes and drawing tools
- Flowchart and diagramming shapes
- Card assignments and due dates
- WIP (Work In Progress) limits
- Automated board rules and triggers
- Real-time cursor tracking
- Voting and team polling features
- Board export to image or PDF
