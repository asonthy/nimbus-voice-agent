# Nimbus SSO — Pricing
Source: pricing

Nimbus SSO Pricing Tiers:
- **Free**: Free
  - SSO for up to 3 apps
  - Basic directory
  - Standard MFA
- **Starter**: $15/user/mo ($12/user/mo billed annually)
  - Unlimited SSO apps
  - Custom login portal
  - Group-based access
- **Professional**: $35/user/mo ($28/user/mo billed annually)
  - Adaptive MFA
  - Automated provisioning (SCIM)
  - Active Directory integration
  - Advanced reporting
- **Enterprise**: Custom / Contact sales
  - Custom SSO connections
  - Advanced security policies
  - SLA guarantees
  - Dedicated support
Add-ons:
- Advanced Identity Protection: $8 / user / month — Adds machine learning driven risk scoring to prevent unauthorized access attempts.
